📁 Куда сохраняется файл на iPhone
После нажатия «Экспорт» Safari предложит сохранить файл. Выберите «Файлы» → папку «На моём iPhone» или iCloud Drive. Файл будет там же лежать для импорта. Рекомендуем создать папку «Wardrobe» для удобства.
Экспорт данных
Все вещи, луки, фото → JSON-файл
Импорт данных
Восстановить из файла резервной копии
Авто-сохранение
Данные сохраняются автоматически в браузере при каждом изменении

При переустановке ярлыка на iPhone
